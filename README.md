# Gemini CLI 원클릭 키트 (초보자용)

> 🌐 **한국어** | [English](README_EN.md)

Google의 AI **Gemini**를 내 컴퓨터(Windows)에서 검은 창(터미널)으로 바로 쓰게 해주는 **초보자용 도구**입니다.
어려운 설정 없이 **파일 하나를 더블클릭 → 숫자 키 누르기**가 전부입니다.

> 📌 이 키트는 Google의 공식 도구 `@google/gemini-cli`를 **더 쉽게 깔고 쓰도록 도와주는 비공식 안내 런처**입니다.
> Gemini AI 자체는 Google이 만들고 운영합니다. 이 키트는 Google과 무관하며 Google의 보증을 받지 않습니다.

> ⚠️ **2026년 6월 18일부터 '구글 로그인'만으로 쓰던 무료 방식이 종료되었습니다.**
> 이제는 **무료 'API 키'**(이 문서가 발급법을 안내합니다)를 한 번 넣으면 그대로 쓸 수 있습니다.

---

## ⚡ 빠른 시작 (4단계, 약 5~10분)

1. **다운로드**: `Gemini-CLI_One-Click_Kit.bat` 파일을 내 컴퓨터의 한 폴더에 둡니다.
2. **설치**: 그 파일을 더블클릭 → 메뉴에서 **[1] 설치**.
3. **API 키 설정**: 메뉴에서 **[2] API 키 설정** → 무료 키를 붙여넣기. (발급법은 아래 4번 단계 설명)
4. **실행**: 메뉴에서 **[3] 실행** → 검은 창에 한국어로 질문을 적으면 끝.

> 숫자 키는 **하나만 누르면 바로** 작동합니다. Enter를 누를 필요가 없습니다.

| 항목 | 설명 | 비용 |

## 2. 다운로드 방법

- `Gemini-CLI_One-Click_Kit.bat` 파일 **하나만** 있으면 됩니다.
- 같이 들어있는 문서(README, GUIDE, LICENSE, NOTICE)는 안내용이라 실행에는 필요 없습니다.
- 파일을 **바탕화면**이나 **내 문서** 같은 찾기 쉬운 폴더에 두세요.

> 💡 받은 파일을 더블클릭했을 때 파란 경고창("Windows의 PC 보호")이 뜨면 → **5. 문제·오류 대처**를 보세요. 바이러스가 아니라 정상 경고입니다.

## 3. 설치 방법 (메뉴 1)

1. `Gemini-CLI_One-Click_Kit.bat` 를 **더블클릭**합니다.
2. 검은 창에 메뉴가 뜨면 **[1]** 키를 누릅니다.
3. Node.js가 없으면 "자동 설치할까요?"라고 물어봅니다 → **Y**.
4. "설치 완료!"가 나오면 끝. (보통 30초~1분)

> Node.js를 방금 자동 설치했다면, **창을 닫았다가 다시 더블클릭**한 뒤 [1]을 한 번 더 누르세요(경로 갱신).

---

## 4. API 키 설정 (메뉴 2) — 무료 키 한 번만

**왜 필요한가요?** 2026-06-18부터 구글 로그인 무료 방식이 막혔습니다. 대신 무료 **API 키**를 쓰면 됩니다. 한 번만 넣으면 키트가 기억합니다.

**무료 키 만들기 (신용카드 필요 없음)**
1. 브라우저에서 👉 <https://aistudio.google.com/apikey> 접속 → 구글 로그인
2. **[Create API key]**(API 키 만들기) 클릭
3. 만들어진 키 옆 **복사** 버튼 클릭 (키는 보통 `AIza…`로 시작하는 긴 글자)

**키트에 넣기**
1. 키트 메뉴에서 **[2] API 키 설정**
2. 안내가 나오면, 복사한 키를 **마우스 오른쪽 클릭**으로 붙여넣고 **Enter**
3. "저장 완료!"가 나오면 끝. 이제 [3] 실행/[4] 한 줄 질문이 바로 됩니다.

> 🔒 **키는 비밀번호와 같습니다.** 남에게 보여주거나 채팅·게시판에 붙여넣지 마세요.
> 키트는 키를 이 PC의 사용자 환경변수 `GEMINI_API_KEY`에 저장합니다(인터넷으로 보내지 않음).
> 키를 지우려면 **[8] 제거 → "API 키도 삭제"**.

---

## 5. 실행 방법 (메뉴 3) & 사용 방법

1. 키트 메뉴에서 **[3] 실행**.
2. 처음에 **"이 폴더를 신뢰하시겠습니까?(trust)"** 라고 물으면 → **신뢰/Yes**를 고르세요.
3. 대화창이 열리면 **그냥 한국어로** 적으면 됩니다. 외울 명령어 없음.
   - 예) `오늘 점심 뭐 먹지?` / `정중한 사과 이메일 써줘` / `파이썬으로 구구단 코드 짜줘`
4. 대화를 끝내려면 입력창에 `/quit` 또는 **Ctrl+C 두 번**.

**한 줄 질문 (메뉴 4)**: 대화창에 들어가지 않고 질문 하나만 빠르게 묻고 답을 받습니다.

---

## 6. 작동 방법 (이 키트가 하는 일)

이 키트는 **복잡한 명령어를 대신 입력해 주는 안내 런처**입니다.

```
[당신] 숫자 키 누르기
   │
   ▼
[키트(.bat)] 올바른 명령을 대신 실행
   │   - 설치:  npm install -g @google/gemini-cli
   │   - 키설정: 환경변수 GEMINI_API_KEY 저장
   │   - 실행:  gemini  (대화)  /  gemini -p "질문" (한 줄)
   ▼
[Gemini CLI] → 인터넷 → [Google Gemini AI] → 답변
```

즉, 키트는 화면을 한국어로 쉽게 보여주고 명령을 대신 쳐줄 뿐, **실제 AI 계산은 Google 서버에서** 일어납니다.

## 7. 메뉴 전체 설명

> 숫자 키 **하나만** 누르면 바로 이동합니다. Enter 불필요.
> 💡 키트가 메뉴 맨 위에 **"지금 누를 것"** 을 내 상태에 맞춰 자동으로 알려줍니다 — 그대로 따라 누르면 됩니다.

| 번호 | 기능 | 설명 |
|---|---|---|
| **1** | 설치 ⭐ | Gemini를 내 컴퓨터에 깝니다 (처음 한 번) |
| **2** | API 키 설정 ⭐ | 무료 키를 한 번 저장합니다 |
| **3** | 실행 ⭐ | AI와 대화를 시작합니다 |
| **4** | 한 줄 질문 | 대화창 없이 질문 하나만 빠르게 |
| **5** | 사용법 | 따라하기 + 예시·명령어 |
| **6** | 자가진단 | 잘 됐는지 확인 / 문제 원인 찾기 (결과를 파일로 저장) |
| **7** | 업데이트 | 최신 버전으로 올립니다 |
| **8** | 제거 | 깔끔하게 지웁니다 (API 키·설정도 선택 삭제) |
| **0** | 종료 | 창을 닫습니다 |

⭐ = 처음 사용자가 차례로 누르면 되는 단계.

## 8. 명령어 모음

**대화창 안에서 (메뉴 3 실행 중)**
| 명령 | 뜻 |
|---|---|
| `/help` | 도움말 보기 |
| `/clear` | 화면 정리 (또는 Ctrl+L) |
| `/quit` | 대화 끝내기 (또는 Ctrl+C 두 번) |

**직접 입력하는 명령 (키트 없이 쓰고 싶을 때, 참고용)**
| 명령 | 뜻 |
|---|---|
| `npm install -g @google/gemini-cli` | 설치 |
| `npm install -g @google/gemini-cli@latest` | 업데이트 |
| `npm uninstall -g @google/gemini-cli` | 제거 |
| `gemini` | 대화 시작 |
| `gemini --skip-trust -p "질문"` | 한 줄 질문 |

## 9. 문제·오류 대처 방법

**먼저: 메뉴 [6] 자가진단을 누르세요.** 무엇이 문제인지 한눈에 보여주고, 키트 폴더에 `gemini_jindan.txt`(진단 결과) 파일로도 저장합니다.

| 증상 | 이렇게 하세요 |
|---|---|
| 파란 경고창 **"Windows의 PC 보호"** | **[추가 정보] → [실행]**. (또는 파일 우클릭 → [속성] → 아래 **[차단 해제]** 체크 → [확인] 후 다시 더블클릭) 인터넷에서 받은 파일이라 뜨는 정상 경고입니다. |
| **"Node.js가 없습니다"** | **[1] 설치**가 자동으로 깔아줍니다. 안 되면 <https://nodejs.org> LTS 설치 후 창을 다시 엽니다. |
| 설치 실패 | 인터넷 연결 확인 후 **[1] 설치** 다시. |
| **"명령을 못 찾음"** / 방금 깔았는데 안 됨 | 검은 창을 **닫았다가 다시 더블클릭**(경로 갱신). |
| 실행했더니 **키가 없다고 함** | **[2] API 키 설정**을 먼저 하세요. 방금 다른 창에서 넣었으면 창을 닫고 다시 여세요. |
| **"trusted directory" / 폴더 신뢰** 물음 | 대화 모드면 **신뢰/Yes** 선택. 한 줄 질문은 키트가 자동 처리합니다. |
| 답이 안 나오고 오류 | 키가 올바른지(복사 실수 없는지) 확인 후 **[2]**에서 다시 넣기. |

## 10. 워크플로우 (전체 흐름)

```
다운로드 → [1] 설치 → [2] API 키 설정 → [3] 실행 → 한국어로 질문 → 답변
                                   └→ [4] 한 줄 질문 (빠른 1회)
문제가 생기면 → [6] 자가진단
새 버전 나오면 → [7] 업데이트
그만 쓰려면 → [8] 제거
```

## 12. 제거 방법 (메뉴 8)

메뉴 **[8] 제거**를 누르면 차례로 물어봅니다:
1. 프로그램(Gemini)을 지울지
2. 저장한 **API 키**(`GEMINI_API_KEY`)도 지울지
3. 설정 폴더(`~/.gemini`)도 지울지

직접 지우려면: `npm uninstall -g @google/gemini-cli`

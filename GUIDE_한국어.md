# Gemini CLI 원클릭 키트 — 왕초보 가이드 (한국어)

이 가이드는 **컴퓨터·AI·프로그램이 처음인 분**을 위해, 화면에 무엇이 보이고 무엇을 눌러야 하는지 하나하나 설명합니다.
빠르게만 보고 싶으면 `README.md`를 보세요. 이 문서는 **천천히, 끝까지** 따라가는 버전입니다.

---

## 0. 먼저 알아둘 쉬운 용어 (1분)

| 용어 | 쉬운 뜻 |
|---|---|
| **터미널 / 검은 창** | 글자로 명령을 주고받는 까만 창. 이 키트가 자동으로 띄워줍니다. |
| **Gemini (제미나이)** | Google이 만든 무료 AI. 질문하면 답해 줍니다. |
| **Gemini CLI** | 그 AI를 검은 창에서 쓰게 해주는 Google의 공식 프로그램. |
| **이 키트(.bat)** | 위 프로그램을 **쉽게 깔고 쓰게** 도와주는 한국어 안내 도구. |
| **Node.js (노드)** | Gemini가 돌아가려면 필요한 '기반 엔진'. 키트가 자동으로 깔아줍니다. |
| **API 키** | 무료 AI를 쓰기 위한 '열쇠 글자'. 한 번만 넣으면 됩니다. |
| **환경변수** | 컴퓨터가 기억하는 설정값. 키트가 여기에 키를 저장합니다. |
| **더블클릭** | 마우스 왼쪽 버튼을 빠르게 두 번 누르기. |

> 모르는 단어가 나와도 괜찮습니다. 아래 단계만 그대로 따라 하면 됩니다.

## 1. 시작 전에 (사전 준비물)

- **Windows 10 또는 11** 컴퓨터
- **인터넷 연결** (와이파이 또는 랜선)
- 그게 전부입니다. Node.js와 API 키는 키트가 도와줍니다.

---

## 2. 파일 준비 & 첫 실행

1. `Gemini-CLI_One-Click_Kit.bat` 파일을 **바탕화면** 같은 찾기 쉬운 곳에 둡니다.
2. 그 파일을 **더블클릭**합니다.

### 파란 경고창이 떴나요?
"**Windows의 PC를 보호했습니다**" 라는 파란 창이 뜰 수 있습니다.
→ 바이러스가 아니라, 인터넷에서 받은 파일이라 뜨는 **정상 경고**입니다.
1. **[추가 정보]** 글자를 클릭
2. 아래에 생기는 **[실행]** 버튼 클릭

> 다른 방법: 파일에서 **마우스 오른쪽 클릭 → [속성] →** 창 아래쪽 **[차단 해제]** 체크 → **[확인]** → 다시 더블클릭.

### 검은 창과 메뉴
잠시 뒤 까만 창이 뜨고 이런 메뉴가 보입니다:

```
    1.  설치          처음 한 번만        ( 추천 1단계 )
    2.  API 키 설정   무료 키 한 번 저장  ( 추천 2단계 )
    3.  실행          AI와 대화 시작      ( 추천 3단계 )
    ...
```

> 숫자 키는 **하나만 누르면** 바로 작동합니다. Enter는 안 눌러도 됩니다.
> 💡 메뉴 맨 위에 **"지금 누를 것"** 줄이 있습니다. 키트가 내 컴퓨터 상태를 보고
> 다음에 누를 번호( [1] → [2] → [3] )를 콕 짚어 줍니다. **헷갈리면 그 줄만 따라 누르세요.**

## 3. [1] 설치 — 처음 한 번만

1. 키보드에서 **`1`** 을 누릅니다.
2. Node.js가 없으면 "**지금 자동 설치할까요? Y=예 / N=아니오**" 라고 물어봅니다.
   → **`Y`** 를 누릅니다. (1~3분 걸릴 수 있어요. 창이 멈춘 듯해도 기다리세요.)
3. 이어서 Gemini를 내려받아 깝니다. "**설치 완료!**" 가 나오면 성공입니다.
4. 아무 키나 눌러 메뉴로 돌아갑니다.

### 만약 "창을 닫았다가 다시 열라"는 안내가 나오면
Node.js를 방금 깐 경우입니다.
1. 검은 창을 **X로 닫습니다.**
2. `Gemini-CLI_One-Click_Kit.bat` 를 **다시 더블클릭.**
3. 메뉴에서 **`1`** 을 한 번 더 누릅니다. → 이번엔 끝까지 설치됩니다.

---

## 4. [2] API 키 설정 — 무료 열쇠 한 번 넣기

> **왜 필요한가요?** 2026년 6월 18일부터 '구글 로그인'만으로 쓰던 무료 방식이 막혔습니다.
> 대신 무료 **API 키**를 한 번 넣으면 됩니다. (신용카드 필요 없음)

### (가) 무료 키 만들기 — 브라우저에서
1. 인터넷 브라우저(크롬/엣지)를 엽니다.
2. 주소창에 다음을 입력해 접속: **aistudio.google.com/apikey**
3. 구글 계정으로 **로그인**합니다.
4. **[Create API key]**(또는 [API 키 만들기]) 버튼을 클릭합니다.
5. 만들어진 키 옆의 **[복사]** 버튼을 클릭합니다.
   - 키는 보통 `AIza…` 로 시작하는 긴 글자입니다.

### (나) 키트에 넣기
1. 키트 메뉴에서 **`2`** 를 누릅니다.
2. "**키를 붙여넣고 Enter**" 안내가 나오면:
   - 검은 창 안에서 **마우스 오른쪽 버튼을 한 번 클릭**하면 붙여넣기가 됩니다.
   - (Ctrl+V가 아니라 **오른쪽 클릭**입니다.)
3. **Enter** 를 누릅니다.
4. "**저장 완료!**" 가 나오면 끝입니다.

### 🔒 키 안전 수칙 (중요)
- 키는 **비밀번호와 같습니다.** 남에게 보여주거나 채팅·게시판·이 대화에 붙여넣지 마세요.
- 무료 등급은 입력 내용이 AI 개선에 쓰일 수 있으니, **개인정보·회사 자료를 넣지 마세요.**
- 키가 노출되면 aistudio.google.com/apikey 에서 그 키를 **삭제(휴지통)** 하고 새로 만드세요.

## 5. [3] 실행 — AI와 대화하기

1. 키트 메뉴에서 **`3`** 을 누릅니다.
2. 안내를 읽고 아무 키나 눌러 시작합니다.
3. 처음이라면 "**이 폴더를 신뢰하시겠습니까? (trust)**" 라고 물을 수 있습니다.
   → 화살표 키로 **신뢰(Trust / Yes)** 를 골라 Enter. (내 폴더이니 안전합니다.)
4. 대화창이 열리면, **그냥 한국어로** 적고 Enter를 누르면 됩니다.

### 이런 걸 물어보세요
- `오늘 점심 메뉴 3개만 추천해줘`
- `아래 문장 맞춤법 고쳐줘:` (그다음 줄에 문장 붙여넣기)
- `정중한 사과 이메일 써줘`
- `엑셀에서 합계 구하는 방법 알려줘`
- `파이썬으로 구구단 코드 짜줘`
- `이 영어 문장 한국어로 번역해줘`

### 대화 중 명령
- `/help` → 도움말
- `/clear` → 화면 정리 (또는 Ctrl+L)
- `/quit` → 대화 끝내기 (또는 Ctrl+C 두 번)

## 6. [4] 한 줄 질문 — 빠른 1회 질문

대화창에 들어가지 않고 질문 하나만 빠르게 묻고 싶을 때 씁니다.
1. 메뉴에서 **`4`** 를 누릅니다.
2. 질문을 적고 **Enter**.
3. 답이 나오면 아무 키나 눌러 메뉴로 돌아갑니다.

## 8. [8] 제거 — 깔끔하게 지우기

메뉴에서 **`8`** 을 누르면 차례로 물어봅니다:
1. "정말 제거할까요?" → **Y** 하면 Gemini 프로그램을 지웁니다.
2. "API 키도 삭제할까요?" → 원하면 **Y** (저장한 열쇠를 지웁니다).
3. "설정 폴더도 지울까요?" → 원하면 **Y**.

> 실수로 8을 눌러도 괜찮습니다. **항상 한 번 더 물어보므로** N을 누르면 취소됩니다.

## 9. 자주 묻는 질문 (FAQ)

**Q. 정말 무료인가요?**
A. 키트와 Gemini CLI는 무료입니다. 무료 API 키도 신용카드 없이 만듭니다. 단, 무료 한도는 Google이 정하며 변동될 수 있습니다.

**Q. 검은 창이 무서워요. 안전한가요?**
A. 검은 창은 그냥 글자로 대화하는 창입니다. 키트는 표준 명령만 실행하며, 코드는 `_build_kit.txt`/`.bat`에서 그대로 볼 수 있습니다.

**Q. 실행했더니 "키가 없다"고 나와요.**
A. **[2] API 키 설정**을 먼저 하세요. 방금 다른 창에서 넣었다면, 창을 닫고 키트를 다시 여세요(저장한 키는 새 창에서 인식됩니다).

**Q. "trusted directory" 같은 영어가 나와요.**
A. "이 폴더를 믿어도 되나요?"라는 보안 질문입니다. 내 폴더이니 **신뢰/Yes**를 고르세요. 한 줄 질문(메뉴 4)은 키트가 자동 처리합니다.

**Q. 답이 안 나오고 오류가 떠요.**
A. ① 인터넷 확인 ② 키가 올바른지(복사 실수 없는지) 확인 후 [2]에서 다시 넣기 ③ [6] 자가진단 결과를 확인.

**Q. 다른 컴퓨터에서도 쓰려면?**
A. 그 컴퓨터에서도 [1] 설치 → [2] API 키 설정을 다시 하면 됩니다. (키는 컴퓨터마다 따로 저장됩니다.)

**Q. 개인정보를 입력해도 되나요?**
A. ❌ 무료 등급은 입력이 AI 개선에 쓰일 수 있으니, 개인정보·회사 기밀은 넣지 마세요.
